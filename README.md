项目说明（MOT-Analysis-GUI）
=================================================
## 描述
### 该项目属于毕业课题内的一部分内容，目的是基于 PyQt5 设计一个简单的软件，实现无人机检测和跟踪可视化功能，其中主要将课题所改进的 YOLO 目标检测算法及相关的 YOLO 系列算法集成在一个可视化界面中，此外，软件还集成了 Deep Sort 跟踪算法，可以实现对无人机检测和跟踪本地实验分析和可视化
